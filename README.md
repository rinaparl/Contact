# Contacts App

Aplikasi sederhana untuk menyimpan daftar kontak menggunakan Flutter dan SharedPreferences.

## Fitur

- Menambahkan kontak baru
- Mengedit kontak yang ada
- Menghapus kontak
- Penyimpanan persisten menggunakan SharedPreferences
